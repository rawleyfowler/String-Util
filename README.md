# String::Util

`String::Util` provides a collection of small, handy utilities for processing strings.

## Installation

    cpanm String::Util

or

    perl Makefile.PL
    make
    make test
    make install

## Usage

See [documentation](https://metacpan.org/pod/String::Util)
